# Property Search AI Chat Assistant

This is a property search chatbot that helps users search for properties across multiple data sources including DORIS, DLR, CERSAI, and MCA21.

## Features

- Natural language property search
- Integration with multiple property data sources
- Modern and responsive UI
- Real-time property information
- Contact support options

## Setup Instructions

1. Extract all files from the ZIP archive to a directory of your choice
2. Open `chatbot.html` in a web browser
3. The chatbot will automatically initialize and appear after 3 seconds

## File Structure

- `chatbot.html` - Main HTML file with UI and styles
- `chatbot.js` - Chatbot implementation and logic
- `propertyService.js` - Property search service and API connectors

## Usage

The chatbot can understand queries like:
- "Search for properties in Maharashtra"
- "Find property ID ABC123"
- "Look for commercial properties in Bangalore"
- Type "help" for assistance
