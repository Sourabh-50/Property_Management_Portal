Accessibility Statement

Our commitment to digital accessibility for all

1. Our Commitment

Unified Property Input is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

Key Principles

  • Equal access to information and functionality
  • Consistent navigation and structure
  • Clear and readable content
  • Compatible with assistive technologies

2. Accessibility Features

  • Screen reader compatibility
  • Keyboard navigation support
  • Text resizing without loss of functionality
  • High contrast mode support
  • Alternative text for images
  • Clear heading structure
  • Consistent navigation mechanisms
  • Form labels and error identification

3. Supported Technologies

  • Latest versions of screen readers (NVDA, JAWS, VoiceOver)
  • Modern web browsers with accessibility features
  • Operating system accessibility tools
  • Mobile device accessibility features

4. Known Issues

We are working to resolve the following accessibility issues:

  • Some older PDF documents may not be fully accessible
  • Some complex maps may require alternative formats
  • Legacy content being updated for accessibility

5. Feedback and Support

We welcome your feedback on the accessibility of our website. Please let us know if you encounter any accessibility barriers:

Email: accessibility@property.gov.in

Phone: +91-XXX-XXXX-XXXX

6. Standards Compliance

Our website aims to comply with:

  • Web Content Accessibility Guidelines (WCAG) 2.1 Level AA
  • Section 508 of the Rehabilitation Act
  • Indian Government Accessibility Guidelines
